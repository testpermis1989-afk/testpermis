================================================
  PERMIS MAROC - OUTIL D'ACTIVATION
================================================

Outil leger pour activer et suivre les licences
des clients Permis Maroc.

UTILISATION:
-----------
1. Double-cliquer sur BUILD.bat pour compiler
2. Le fichier PermisMaroc-Activation-Setup.exe sera cree
3. Envoyez ce setup aux clients

FONCTIONNALITES:
----------------
CLIENT:
- Voit le Code Machine de son PC
- Copie le code en un clic
- Entre le code d'activation recu

ADMIN (mot de passe: admin123):
- Genere des codes d'activation
- Voit la liste de toutes les licences
- Voit les activations en cours
- Supprime les licences

DUREES DISPONIBLES:
- 30 jours
- 90 jours
- 6 mois
- 1 an
- Illimitee
